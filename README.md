
## Description

E-commerce fullstack applicaiton. Live demo on heroku [https://shoe-jk.herokuapp.com/](https://shoe-jk.herokuapp.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
